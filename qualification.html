  • Abstract: Several mature and high-performance libraries are available today that provide efficient I/O for scientific applications. However, there is very little research done to evaluate and improve I/O performance of Java-based HPC applications. As part of this project, we plan to develop a Java-based parallel I/O API, a reference implementation on top of existing Java messaging libraries and then performance comparison of our reference Java Parallel I/O library with C/C++ counterparts using benchmarks and real-world applications on proprietary networks including Myrinet and Infiniband.
